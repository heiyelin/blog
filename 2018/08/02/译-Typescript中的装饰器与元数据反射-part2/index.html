[译]Typescript中的装饰器与元数据反射(part2)

Posted by Alin on 2018-08-02

这篇文章会深入讲解 Typescript 如何实现装饰器以及诸如反射或依赖注入这些令人激动的 javascript 特性。

这个系列包含了一下几个部分：
第一部分：方法装饰器
第二部分：属性装饰器 & 类装饰器
第三部分：参数装饰器 & 装饰器工厂
第四部分：类型序列化 & 元数据反射 API

在这个系列前面的文章中，我们从Typescript源码中找到了可用的装饰器特性的实现。

我们也学习了怎么实现一个方法装饰器，也回答了在Typescript中装饰器如何工作的几个问题：

  • 装饰器时在哪里被调用的？
  • 谁提供了装饰器参数？
  • __decorate函数在哪里被定义？

如果你看了这个系列第一篇文章，你能回答得上来上面的几个问题。

在这篇文章中，我们将要学习两种装饰器类型：PropertyDecorator属性装饰器 和 ClassDecorator类装饰器。
