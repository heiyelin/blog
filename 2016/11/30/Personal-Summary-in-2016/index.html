Personal Summary in 2016

Posted by Alin on 2016-11-30

在 2016 年的最后几天，总会有许多感概想记录下来，写一写这一年做了什么，又有哪些未完成的遗憾啊。

1

这一年里，可能是最后在学校里学习的时候了吧。一二月还在忙着团队的项目，而且还过了一个很好的年，三四五月就开学了，一直处于在学习的状态，虽缓，但也前行。真正下定决心想往前端方向走的还是在这几个月里吧，因为自己的基础不是很好，很多东西都是要恶补的，对于前端也只是一个入门的初学者。所以在那时候一直在深入学习 javascript 这门难懂的语言，尤其是在面向对象的高级编程方面，学了忘，忘了学，可能就是当时学习的状态吧。

在前端路上的学习上，虽然不是一下子就能提高的，但走得让自己踏实就好。在学习 js 的过程中，自己在 segmentfault 中写了很多的技术日记，希望自己学得更加牢固。学完了 js，接着就是 jquery，学习怎么编写一个 jquery 插件，再到 js 的模块化，学习利用 requirejs 进行 js 模块化，再到 nodejs 的入门，学习怎么搭建一个简易的静态服务器，到自己照猫画虎的做个个人博客。后来，觉得自己应该要有个可以写技术日记的地方，就买了个域名，用 hexo 做起了这个博客，虽然简陋，但是有个可以写字的地方还是不错的。

暑假的时候去了广州，学校安排的生产实训。后来在网上投了一些简历，但也有碰壁，所以觉得自己的能力还是不够，所以那时候就毅然回家学习。也是在这一两个月的学习中，让自己更加对前端有了更大的兴趣，接触了 gulp、webpack 这些前端自动构建工具，还有前端工程化、前端组件化的思想，例如 react 等。回到学校后，继续学习着前端，学习 nodejs 和 react 以及 webpack，学习 es6 的一些新特性。

这一年，前端变化得太快。但是，我一直觉得基础很重要，如果连所谓的基础知识都掌握不好，怎么可能把各种新技术学好，估计也只是会用，却不懂为什么这么用。这也是我在学习的过程中常常感到困惑的地方。

2

一直觉得很多东西都不应该只是想想而已，更多的要去做。想得太多，也比不上开始做的那一刻。很多事情不做，就会留下遗憾。不断地希望自己努力变得更好，却忘记怎么去开始改变。

这一年的冬天，拍完毕业照的那刻，就觉得自己已经毕业了，已经过了那个无忧无虑的年纪，要开始懂事了，未来有着更多的责任需要去承担。惶恐不安、迷惘的心情大概都是即将离校的人有的共同感受吧。

太多事情想做，却没去做，这大概是自己这一年最大的遗憾。

3

2017，还有一两天。写这么多只是想发发牢骚，停下来看看过去的自己，也希望在美好的 2017 这一年里，自己要更加专注、踏实地前行，努力过上自己想要的生活。同时，要更加更加珍惜身边的人，无论是亲人还是朋友。
